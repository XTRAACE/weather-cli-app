# 🌦️ Weather CLI App

A simple and stylish **command-line weather application** built with Python.  
It fetches real-time weather data using an API and displays it with colored output and a big ASCII header.

---

## 🚀 Features

- 🌍 Get real-time weather by city name  
- 🎨 Colored terminal output using Colorama  
- 🔤 Big ASCII header using PyFiglet  
- 🔐 Secure API key using `.env`  
- ⚡ Fast and lightweight CLI tool  

---

## 🛠️ Tech Stack

- Python  
- requests  
- colorama  
- pyfiglet  
- python-dotenv  

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-cli-app.git
cd weather-cli-app
Install dependencies:
pip install -r requirements.txt
🔑 Setup API Key

This app uses WeatherAPI

Sign up and get your API key
Create a .env file in the root directory:
API_KEY=your_api_key_here
▶️ Run the App
